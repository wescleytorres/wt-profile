## Getting started

## - Instalando Dependencias

No diretório web inserir o comando:

NPM:
``` bash
npm install
```
YARN:
``` bash
yarn OU yarn install
```

## - Rodando aplicação

No diretório web inserir o comando:

NPM:
``` bash
# http://localhost:3000/
npm run start
```
YARN:
``` bash
# http://localhost:3000/
yarn start
```
